CSS Position & Display
Cheatsheet

Position

Positioning allows a user options on where to set elements on a webpage.

Value Description
Static element {
position: static; }		 The element is positioned according to the normal flow of the document. This is the default value.
Relative element {
position: relative; }		 The element is placed in it's original positon and offset properties can be used to determine it's actual position.
Absolute element {
position: absolute; }		 Enables an element to ignore sibing elements and be positioned relative to it's parent element.
Fixed element {
position: fixed; }		 The element is set to a specific spot regardless of scrolling.
Sticky element {
position: sticky; }		 The element can scroll to an offset value provided by the user.

Display

Determines the display behavior of an element.

Value Description
Block container {
display: block; }		 Elements will take up the full width of their container with line breaks before and after. Height and width can be adjusted.
Inline container {
display: inline; }		 Elements will take up as little space as possible and flow horizonally. Height and width cannot be adjusted.
Inline Block container {
display: inline-block; }		 Elements appear ext to each other and can have their height and width adjusted.
